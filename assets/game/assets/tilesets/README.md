# Tilesets

Place tile atlas images here for the overworld and interiors.

| File | Description | Usage |
|------|-------------|--------|
| **exterior-tileset.png** | Town, grass, paths, buildings | Loaded as `exterior-tileset`. Used for overworld (WorldScene). Tile size should match game (e.g. 32Ã—32). |
| **interior-tileset.png** | Inside Matt's Home + Studio | Loaded as `interior-tileset`. Used for interior scenes. |

If these files are missing, the game falls back to Graphics-generated placeholder tiles.
